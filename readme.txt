== Description ==

Displays an AboutMe bloc, including a picture and some text.

== Installation ==

1. Upload the zipfile to the `/wp-content/plugins/` directory
2. Extract and remove it
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Drag the new widget to desired sidebar, choose a title (or not) for the widget and fill additional properties.
